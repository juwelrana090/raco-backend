# 📋 Todo Tasks

Anyone on the team can add tasks here.
Anyone on the team can pick up any task.

## How to add a task

Run: /r-add-task [description]
Or create: .claude/tasks/todo/2026-07-10-[title].md

## Task template

---

# Task: [title]

**Added By**: [name]
**Date Added**: 2026-07-10
**Priority**: 🔴 High / 🟡 Medium / 🟢 Low
**Module**: [which module]
**Estimated Effort**: Small / Medium / Large
**Assigned To**: unassigned

## Description

[what needs to be done and why]

## Acceptance Criteria

- [ ] [what done looks like]

## Notes

[context, gotchas, links]

## Related Files

[relevant file paths]
---

## How to pick up a task

Run: /r-pickup
Commit and push so team knows it is taken.
